pstree PRETTY EXPLORER

This tool transforms the flat text output of Volatility's pstree plugin into an interactive "process hacker" style visualization.

In the tool folder you can find a -r pretty pstree output, the script and the result.

What is the pstree plugin?

The pstree plugin is a plugin that print the processes with their command lines, path, offset, PID, PPID, etc. one important element of the pstree is that he noted the relationship between the different processes giving a tree like view of the child/parent relationship.

The issue is that it's very hard to read the output in the terminal! This is why I created this plugin, the goal is to have a better visualisation of the processes and their realtionship while giving the opportunity to filter and/or exclude certain line.

How to Use the Tool

01. Extract the pstree Data: Run the pstree plugin with the -r pretty option (ATTENTION IF YOU DON'T USE THIS OPTION THE SCRIPT WILL NOT PARSE THE DATA!!!!!) on your memory image and save the output to a text file:

vol -f mem.raw -r pretty windows.pstree > vad_output.txt

OPTIONAL Modify the script: This script is applying a differnt style for suspicious processes name (bold and red), you can add your own suspicious string in the script before launching, for that modify the line 181 const isMal = p.name.toLowerCase().includes('legit') || p.name.toLowerCase().includes('powershell') || p.name.toLowerCase().includes('not_a_virus');, like you can see for the moment 3 strings are suspicious, 'legit', 'not_a_virus' and 'powershell'

02. Run the Python Parser: Use the provided Python script (here) to parse the pstree.txt. The script will parse the pstree output and generate a .html file.

03. Visualize: Open the HTML file with your browser.

04. Filter: In the tool you have two box, one for filtering the ImageFileName and one to exclude processes based on the cmd column.

Why Visualize the pstree?

  • SIMPLICITY A lot easier to read than in the terminal.
  • FUNCTIONNALITY: Use the filter box and the exclusion box to keep only what you want to see!
  • MALWARE ANALYSIS: Spot the weird process easily!
