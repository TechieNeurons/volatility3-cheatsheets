VAD Tree Visualizer

This tool transforms the flat text output of Volatility's vadwalk and/or vadinfo plugin into an interactive Binary Search Tree (BST) visualization.

In the tools folder you can find a vadwalk output, vadinfo output, the script and the result.

What is the VAD?

The Virtual Address Descriptor (VAD) is a data structure used by the Windows kernel to manage a process's memory address space. It is organized as a self-balancing binary tree to allow the kernel to quickly find memory ranges.

How to Use the Tool

Prerequisite Install the required python modules

pip install pandas plotly networkx

01. Extract VAD Data: Run the vadwalk plugin on your memory image and save the output to a text file:

vol -f mem.raw windows.vadwalk --pid  > vad_output.txt

01 (bis). Extract VAD Data: Run the vadinfo plugin on your memory image and save the output to a text file:

vol -f mem.raw windows.vadinfo --pid \ > vadinfo_output.txt

02. Run the Python Parser: Use the provided Python script (here) to parse the vad_output.txt. The script identifies the Parent, Left, and Right pointers for each memory range.

03. Visualize: The script generates an HTML file that you can open with your browser.

Why Visualize the VAD?

  • Malware Detection: Easily spot "unbalanced" trees or suspicious memory ranges that don't follow standard naming conventions.
  • Memory Analysis: Quickly identify PAGE_EXECUTE_READWRITE (RWX) sections which are often indicators of code injection.
  • Structure Verification: Verify the integrity of the kernel's memory management by seeing the physical links between nodes.
